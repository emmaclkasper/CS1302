Compile: In order to compile the Vehicle System program begin in the Project 2 directory. From this directory the command is $javac -d classes src/edu/uga/cs1302/vehicles/*.java
	 It is imperative that the package name is correct, all the compiled class files will be in the classes directory. 

Execute: To execute the program begin in the same Project2 directory. The command from here is $java -cp classes edu.uga.cs1302.vehicles.VehicleDriver
	 The main method is inside of the VehicleDriver file and when it is run, the user will interact with that file.
